#Markdown的基本语法

Markdown是一种轻量级标记语言，通过简洁的语法即可实现文本排版，以下是常用的基本语法：

一、标题

使用  #  号加空格表示标题级别，共支持六级标题， #  数量越多级别越低：

-  # 一级标题 
-  ## 二级标题 
-  ### 三级标题 
...以此类推至六级标题。

二、字体样式

- 加粗：文字两侧加  **  或  __ ，如  **加粗文本**  显示为 加粗文本。
- 斜体：文字两侧加  *  或  _ ，如  *斜体文本*  显示为 斜体文本。
- 斜体加粗：文字两侧加  ***  或  ___ ，如  ***斜体加粗文本***  显示为 斜体加粗文本。
- 删除线：文字两侧加  ~~ ，如  ~~删除线文本~~  显示为 ~~删除线文本~~。

三、列表

1. 无序列表

使用  - 、 +  或  *  加空格开头，支持嵌套：


- 项目1  
  - 子项目1.1  
- 项目2  


显示效果：

- 项目1
- 子项目1.1
- 项目2。

2. 有序列表

使用  数字+英文句点+空格  开头，自动排序：


1. 第一项  
2. 第二项  


显示效果：

1. 第一项
2. 第二项。

四、引用

使用  >  加空格表示引用，支持嵌套：


> 一级引用  
>
> > 嵌套引用  


显示效果：

一级引用

嵌套引用。

五、分割线

使用  --- （三个减号）或  *** （三个星号），单独占一行即可生成分割线：


---

显示效果：
---。

六、链接与图片

1. 超链接

格式： [链接文字](链接地址) ，如  [百度](https://www.baidu.com)  显示为 百度(https://www.baidu.com)。

2. 图片

格式： ![图片描述](图片地址) ，如  ![示例图片]("D:\Users\Screenshot_2025-08-31-00-19-34-75_149003a2d400f6a.jpg") （图片地址可使用网络链接或本地路径)。

七、表格

使用  |  分隔列， -  分隔表头与内容，语法如下：


| 姓名 | 性别 | 生日       |
| ---- | ---- | ---------- |
| 张三 | 男   | 1990-01-01 |


显示效果：

姓名    性别    生日
张三    男    1990-01-01

提示：部分编辑器需删除表格行之间的空行以正常显示。

八、代码

1. 行内代码

使用  ` （反引号）包裹代码，如  `print("Hello World")`  显示为  print("Hello World") 。

2. 代码块

使用三个反引号 ``` 加语言名称（可选）包裹代码块：


print("Hello Markdown!")  


显示效果：


print("Hello Markdown!")
